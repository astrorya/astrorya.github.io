Machine Learning

Currently, I am working with Dr. Lindsey Bleem at Argonne National Laboratory. My project is focused on using machine learning algorithms to detect Brightest Cluster Galaxies (BCGs) in galaxy clusters. So far, this work has led to the development of Image Marker, a new program designed for quickly identifying and logging features in images from a large dataset. We will be using Image Marker to create a human-truth sample of BCGs in galaxy clusters detected by the SPT-3G camera aboard the South Pole Telescope. I am still working on this project, so once I have more information to share, I will update this page.
